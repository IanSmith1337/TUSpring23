# Cyclomatic Complexity

## Definition

How complex the code is. This number determines how many test cases are needed.

Calculated from a [[Control Flow Graph]].

$$M = E-N+2P$$

## In-Class Example

$$ M = 10 - 9 + 2(1) = 1+2(1) = 3 $$
